手上青秀

手上青秀APP是一款专门用于市民、企业在线办理行政审批事项、社区街道事项的政务服务APP。青秀区政府将通过削减行政审批事项，简化优化办事流程，推行行政审批“一事通办”，为办事群众节省时间和精力，通过使用“手上青秀APP”最大限度地推进网上审批、线上服务、“一次都不用跑”。手上青秀APP还整合了青秀区各个部门、各类渠道宣传信息的出口入口，实现宣传信息统一发布，统一服务。让青秀的百姓与企业全方位的了解我区的政务动态。

为了弥补“最多跑一次”改革中的落地“短板”，让更多群众从中受益。手上青秀APP使用多种宣传手段，通过官网、公众号、微博、APP、网络媒体平台等向办事群众、企业宣传“最多跑一次”的事项流程，告知哪些事项可便捷办理，同时也为行政服务中心提升服务效能、拓宽交流渠道提供保障。让群众进一步形成便捷化办结的意识，助力改革落地见效。手上青秀APP除了实现对外宣传、便民服务之外，还将新建政府内部办公OA，增加移动端功能，方便工作人员随时随地处理公务。
